# Multi Uploader
Multi Uploader - это десктопное приложение на базе Electron + React + Redux, которое будет позволять автоматически загружать короткие видео на несколько платформ (YouTube Shorts, TikTok, Instagram и др.).

## Цель проекта
Создать удобное десктопное приложение для автопостинга коротких видео на несколько платформ одновременно с кастомным интерфейсом и безопасной архитектурой.

### На данный момент проект включает:
 - Electron main process с кастомным окном
 - React frontend с Vite и TypeScript
 - Кастомная панель TitleBar с кнопками minimize, maximize и close
 - Возможность перетаскивания окна за TitleBar
 - Переключение между страницами
 - Подключение аккаунта Google с необходимыми правами
 - Возможность выбрать файл
 - Несколько полей ввода
 - Возможность запостить видео на youtube

### В разработке:
 - Страница загрузки видео

### Стек технологий:
 - Electron - десктопное приложение
 - React + Redux + TypeScript - фронтенд
 - Vite - сборка и быстрый HMR
 - styled-components - стилизация компонентов
 - Electron ContextBridge - безопасная связь между main и renderer процессами

## Статус проекта
 - ✔️ Настройка Electron + React + TypeScript + Vite
 - ✔️ Кастомное окно без системного меню
 - ✔️ TitleBar с drag и кнопками управления окном
 - ✔️ Переключение между страницами
 - ✔️ Подключение аккаунта Google с правами на загрузку видео
 - ✔️ Интеграция с API YouTube для автопостинга
 - Интеграция с API TikTok для автопостинга
